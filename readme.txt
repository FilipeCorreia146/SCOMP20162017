========== readme.txt =============================

Leia atentamente este ficheiro antes de iniciar o seu trabalho nas aulas PL de SCOMP.

Regras:

a) Cada grupo (2 alunos) deve ter o seu repositorio.

b) O nome do repositorio no Bitbucket segue o seguinte padrao: SCOMP1617tttGxx
- ttt deve ser substituido pelo nome da turma. Ex: 2DA
- xx deve ser substituido pelo número do grupo (sempre com 2 digitos). Ex: G01
- Exemplo: SCOMP16172DAG01

c) Estrutura de diretorios na raiz do repositorio.
- A estrutura de diretórios do primeiro nivel já está criada. 
- Existe um diretorio para cada entrega das aulas PL.
- DEVERÁ CRIAR um diretorio para cada módulo das aulas PL.
- Dentro do diretorio de cada módulo DEVERÁ CRIAR um subdiretorio para cada exercicio.
- O nome desse subdiretorio deve iniciar pelas letras MINUSCULAS "ex" e ser seguido do numero que identifica o exercicio. Ex: "ex01"
